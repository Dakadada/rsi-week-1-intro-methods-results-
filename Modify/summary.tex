Short peptides naturally form useful structures like hydrogels for drug delivery or have useful properties for novel biomaterials design. However, designing new materials requires large datasets, and experiment details are stuck inside thousands of papers. Manual data collection (and knowledge gathering) is extremely slow, and standard off-the-shelf AI cannot accurately extract data. This project develops a team of specialized agents to read papers, extract conditions, and map the findings. This system generates clear design rules which allow researchers to rapidly access literature data and predict material properties.
